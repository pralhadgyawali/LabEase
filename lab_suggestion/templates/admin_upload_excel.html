Excel File Format Requirements

Required columns (case-insensitive):

Lab Name - Name of the laboratory
Address - Full address
City - City name
State - State/Province
Zip Code - Postal code
Phone Number - Lab phone
Contact Email - Email address
Contact Phone - Contact number
Test Name - Name of the test
Test Description - Test details
Test Price - Price in Rs.

Note: Each row should contain lab information and at least one test. Multiple tests for the same lab can be in separate rows with the same lab name.
