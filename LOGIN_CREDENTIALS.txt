HOW TO LOAD SAMPLE DATA FOR ADMIN DASHBOARD:
--------------------------------------------
To populate the admin dashboard with sample data (labs, tests, messages):

1. Make sure you're in the project directory
2. Run the following command:
   
   python manage.py load_sample_data

This command will automatically:
- Create admin user (if not exists): admin / admin123
- Create 20 different medical tests with prices
- Create 5 sample labs in Kathmandu, Lalitpur, and Bhaktapur
- Create 14 sample contact messages (to labs and admin)
- Create sample chat messages
- Associate tests with labs

After running, you'll see:
- Statistics cards populated in admin dashboard
- Recent messages section filled
- Recent labs section filled
- All labs table with multiple entries

NOTE: If admin user doesn't exist, the command will create it automatically.
